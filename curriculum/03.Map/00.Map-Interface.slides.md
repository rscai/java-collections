## Inheritence Hierarchy

```puml
@startuml
interface Collection<E>
interface Set<E>
interface List<E>
interface Queue<E>
interface Deque<E>
interface SortedSet<E>
interface Map<K,V> #Aqua
interface SortedMap<K,V>

Collection <|-- Set
Collection <|-- List
Collection <|-- Queue
Collection <|-- Deque

Set <|-- SortedSet

Map <|-- SortedMap
@enduml
```
